# ¿Qué es el UX design? #UX

**UX Design** es el proceso sistemático de **investigar, diseñar, evaluar e iterar** soluciones con el objetivo de optimizar la **experiencia global del usuario** al interactuar con un sistema. Esto busca que sean funcionales, intuitivos, accesibles y agradables.

*UX Design implica:*
- **Investigación de usuarios** (UX Research)
- **Definición de problemas**
- **Arquitectura de información**
- **Prototipado**
- **Evaluación y testing**
- **Iteración continua**

No se trata de “dibujar pantallas”, sino de:
- ***entender necesidades***
- ***diseñar soluciones***
- ***validarlas***

---
# ¿Qué es UI Design? #UI

**UI Design** es el ***proceso de diseñar*** los **elementos visuales e interactivos** de una interfaz, con el objetivo de facilitar la interacción entre el usuario y el sistema.

*UI Design incluye:*
- **Diseño visual** (colores, tipografía, estilos)
- **Layout** (disposición de elementos)
- **Componentes** (botones, inputs, menús)
- **Estados** (hover, error, loading)
- **Consistencia visual**

Es la **materialización visual** del sistema.

---
# UX vs UI #UXvsUI

UX se enfoca en la experiencia completa del usuario, mientras que UI se centra en la interfaz.  

La calidad de la UX se evalúa mediante ***factores de usabilidad*** como:

- ***Adecuado para su uso:*** ¿El sistema proporciona las funcionalidades necesarias?
- ***Aprendizaje fácil:*** Tiempo que se tarda en utilizar el sistema apropiadamente.
- ***Tareas eficientes:*** Número de clics, tiempo para realizar una tarea, etc.
- ***Fácil de recordar:*** Si no se usa con frecuencia, ¿un usuario recordará cómo manejar el sistema?
- ***Satisfacción subjetiva:*** ¿Qué tan satisfecho está el usuario con el sistema?
- ***Fácil de comprender:*** Propone un modelo mental claro.

# Conexiones

El diseño de experiencia de usuario (**UX Design**) no ocurre de manera aislada, sino que se construye a partir de un ***enfoque, herramientas y procesos.***

En este contexto:
- [[UCD (enfoque)]] define el **enfoque**, centrando el diseño en el usuario.
- [[UX Mindset (cómo pensar)]] establece la **forma de pensar**, basada en empatía y adaptación al usuario.
- [[UX disciplines (herramientas)]] representan las **áreas de trabajo** que permiten diseñar la experiencia.
- [[Double Diamond]] estructura el **proceso de diseño**, desde la comprensión del problema hasta la solución final.
