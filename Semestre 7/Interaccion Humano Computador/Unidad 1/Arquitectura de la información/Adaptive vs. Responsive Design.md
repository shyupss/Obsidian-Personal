# Responsive Design (Diseño Responsivo)

Es un diseño **fluido**. Como el agua, se adapta a cualquier tamaño de pantalla de forma continua usando rejillas flexibles.

La ***ventaja*** es que ***un solo código*** sirve para todo (reloj, celular, tablet, PC). Es consistente.

**Ejemplo:** La mayoría de los sitios web modernos que se "estiran" si agrandas la ventana del navegador.
# Adaptive Design (Diseño Adaptativo)

El sistema detecta qué dispositivo tienes y te entrega un diseño **fijo** hecho específicamente para ese tamaño ***(usa "breakpoints").***

La ventaja acá es que se puede optimizar la experiencia al máximo para un hardware específico.

**Ejemplo:** Un sistema de punto de venta (POS) en un restaurante o un quiosco de autoservicio en el aeropuerto. No necesitan ser fluidos, necesitan ser perfectos para esa pantalla exacta.
