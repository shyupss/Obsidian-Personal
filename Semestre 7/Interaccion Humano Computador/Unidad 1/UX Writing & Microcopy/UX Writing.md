# ¿Qué es?

No es "escribir bonito", es escribir para **guiar**.

**Crafting text (Crear texto):** Se refiere a cada palabra que ves en una pantalla: el nombre de un botón, el texto de un campo de formulario, o una notificación.

**Clear, helpful and aligned (Claro, útil y alineado):**
- **Claro:** Sin palabras rebuscadas.
- **Útil:** Debe ayudar al usuario a dar el siguiente paso.
- **Alineado:** Debe sonar como la marca (si es una app de juegos, puede ser divertido; si es de un banco, debe ser serio y seguro).
# ¿Por qué es importante?

- Mejora la usabilidad al reducir la confusión y los errores.
- Mejora la interacción del usuario con llamadas a la acción (CTA) claras.
- Genera confianza y credibilidad a través de mensajes coherentes y útiles.

# Clarity in UX Writing

Utilizamos un lenguaje sencillo, evitando la jerga y la ambigüedad.

Debemos acortar las oraciones y concéntrate en el mensaje principal.

| **Escenario**       | **No decir... (Mal ❌)** | **mejor... (Bien ✅)**               | **¿Por qué?**                                                            |
| ------------------- | ----------------------- | ----------------------------------- | ------------------------------------------------------------------------ |
| **Botón de acción** | Submit                  | Create Account                      | "Submit" es genérico. El segundo le dice el resultado final.             |
| **Error de datos**  | Invalid input           | Password must be at least 8 chars   | El primero frustra; el segundo da la solución al problema.               |
| **Error de página** | Error 404               | Page not found. Check the URL...    | Explica qué pasó y ofrece una salida (volver al Home).                   |
| **Placeholder**     | "Telephone"             | "e.g., +569 1234..."                | El ejemplo visual ayuda al cerebro a procesar el formato rápido.         |
| **Seguridad**       | Session expired         | You've been logged out for security | Explica la _razón_ (seguridad), lo cual genera confianza.                |
| **Estado vacío**    | Nothing to show         | Your playlist is empty. Add songs!  | Motiva al usuario a interactuar en lugar de mostrar una pantalla muerta. |
