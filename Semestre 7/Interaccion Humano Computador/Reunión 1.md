# Contexto
## Qué es Territorio Vivo
Organización que impulsa la regeneración de territorios mediante la integración de:
- Producción sostenible
- Restauración ecológica
- Gestión hídrica
- Captura de carbono

Trabaja con comunidades, actores productivos, sector público y privado.
## Modelo de operación
### Líneas de acción
- Restauración de ecosistemas
- Modelos productivos regenerativos
- Gobernanza territorial
- Capacidades locales
- Finanzas regenerativas
## Tipo de sistema
Sistema socio-técnico :
- Multi-actor
- Basado en datos (ambientales + económicos)
- Dependiente de coordinación territorial

# Hipótesis posible
## 1. Complejidad técnica vs usuarios no expertos
Usuarios necesitan entender datos complejos (agua, carbono, biodiversidad) sin formación técnica.
## 2. Falta de herramientas de coordinación
No existe una plataforma clara que conecte actores territoriales.
## 3. Datos no accionables
Existe información, pero no se traduce en decisiones concretas. Por ende necesitan gestionar todo esto.
